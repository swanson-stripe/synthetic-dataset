üöÄ Ultra-Efficient Integration

Get a complete dashboard with persona switching in just 2 lines of code:

// 1. Include the consolidated kit <script src="https://swanson-stripe.github.io/synthetic-dataset/dashboard-synthetic-data.js"></script> // 2. Initialize and you're done! const kit = new DashboardSyntheticData({ onPersonaChange: (data, persona, stage) => { updateMyDashboard(data); } }); // Access everything instantly kit.getMetrics(); // Business metrics kit.getData('payments'); // Payment data kit.switchPersona('mindora'); // Change business

That's it! Persona selector auto-appears, all data available instantly, future-proof updates!
